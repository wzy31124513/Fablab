CLASS TWO

Laser Cutter & 2D Design

We've designed and cutted a nametag by using Rhinoceros and Laser cutter.

Rhinoceros is a 3D modeling software which is commonly used in industrial design. It also can be used in 2D design.

Using Rhinoceros, I designed a nametag which looks like my mobile phone Jianguo pro with my name on it.

The front of Jianguo Pro was made by two materials, glass and aluminium, to realize the effect on wood, I need to use raster engraving.

I need to set different color to decide where to cut and where to engrave. To make two material's effect, I painted the bottom of my name tag yellow to engrave it.

After cutting, I made a name tag which has the same size with my mobile phone.
