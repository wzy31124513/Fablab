CLASS FIVE

We used Roland SRM-20 to make a PCB this class.

First, we convert the image of the circuit into .rml format on Fab modules.

Then, we need to make the chemical wood flat. But we chose the wrong bit. which caused some problem. But it still can be used.

After sticking the PCB board on the chemical wood, and upload .rml on the machine program, it began to mill.(We use 0.4mm bit to mill the trace, and 0.8mm to make the hole)

Finally, I made a PCB that can be used.
